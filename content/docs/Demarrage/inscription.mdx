Cette section explique comment les nouveaux utilisateurs peuvent s'inscrire sur notre plateforme d'évaluation automatisée d'exercices de bases de données.

## Aperçu de la Page d'Inscription

La page d'inscription permet aux nouveaux utilisateurs de créer un compte en fournissant quelques informations essentielles. L'interface est conçue pour être intuitive et facile à utiliser.

## Processus d'Inscription

### Étape 1: Accéder à la Page d'Inscription

Pour accéder à la page d'inscription, cliquez sur le bouton "S'inscrire" situé dans le coin supérieur droit de la page d'accueil.

### Étape 2: Remplir le Formulaire

Le formulaire d'inscription comprend les champs suivants:

1. **Nom d'utilisateur** - Choisissez un nom d'utilisateur unique qui vous identifiera sur la plateforme
2. **Adresse e-mail** - Fournissez une adresse e-mail valide qui sera utilisée pour la vérification et les communications
3. **Mot de passe** - Créez un mot de passe sécurisé (minimum 8 caractères, incluant des lettres, chiffres et caractères spéciaux)



## Fonctionnalités Spéciales

### Intégration avec les Services Institutionnels

Si vous vous inscrivez avec une adresse e-mail académique reconnue, certaines informations peuvent être pré-remplies et votre statut d'étudiant ou d'enseignant peut être automatiquement validé.

### Options d'Authentification Alternative

Vous pouvez également vous inscrire en utilisant:
- Votre compte Google
- Votre compte Github
- Votre identifiant institutionnel (si applicable)

## Résolution des Problèmes Courants

| Problème | Solution |
|----------|----------|
| Nom d'utilisateur déjà pris | Essayez une variante ou utilisez une combinaison plus unique |
| Erreur lors de la soumission du formulaire | Assurez-vous que tous les champs sont correctement remplis et respectent les formats requis |
