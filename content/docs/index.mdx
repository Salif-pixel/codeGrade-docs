# Introduction

Bienvenue sur la documentation de <span className="bg-gradient-to-r from-emerald-300 to-emerald-400 bg-clip-text text-transparent font-bold">CodeGrade</span>, votre plateforme intelligente d'évaluation de code et de travaux académiques. Notre système utilise des technologies d'IA avancées pour automatiser l'évaluation, fournir des retours détaillés et améliorer l'expérience d'apprentissage.

## Pourquoi utiliser CodeGrade ?

Notre plateforme révolutionne l'évaluation de code et de travaux académiques en offrant :

✅ **Gain de temps considérable** - Réduisez jusqu'à 90% le temps de correction
✅ **Évaluation cohérente** - Garantissez une notation uniforme pour tous les étudiants
✅ **Feedback intelligent** - Offrez des retours personnalisés et instructifs
✅ **Flexibilité maximale** - Évaluez trois types de travaux : QCM, code, et document
✅ **Intégration facile** - Déployez rapidement dans votre environnement éducatif existant
✅ **Personnalisation avancée** - Adaptez les critères d'évaluation à vos besoins spécifiques
